# TickTakGame
This is an iOS TicTacToe Game using Swift 5.2

The Brain of the Game is that it moves to State (O or X) at Coordinates (x,y). 
The Coordinates(x,y) are a grid of 3X3 matrix like so below : 


        0     1     2
   ____________________

0    | 0,0  | 1,0  | 2,0 
   ____________________

1    | 1,0  | 1,1  | 1,2   
   ____________________

2    | 2,0  | 2,1  | 2,2
